Super-Newbie Coder's
GUIDE TO
Hacktoberfest

If you're new to coding then Hacktoberfest can seem overwhemling the first time you take part. This website is here to help you if you don't know where to start and are feeling frustrated trying to find repos and issues which are easy enough to work on, even for first-timers. This is a crowd sourced project by Hacktoberfest Newbies by Hacktoberfest newbies.
